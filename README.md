# 프로젝트 소개
플레이어는 정해진 맵에서 Goal지점까지 도착할떄까지 얼마나 빠르게 도달하는지 기록하고 경쟁하는 게임입니다.

#  개발자 소개
개발자: 여현빈

#  프로젝트 계기
전부터 만들고 싶은 3인칭 3D게임을 만들고 시다 생각해서 지금 내가 만들수 있는것과 배우고 싶은 것을 생각하다 플레이어 캐릭터의 자유로운 이동을 만들고 싶었습니다.

# 주요기능
* 플레이어 이동
* 랭킹시스템
* 캐릭터 선택
* 타이머 기능
* 사다리 기능
* 부스트 아이템
